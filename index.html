    3D Can crusher

    Designed and built a mechanical can crusher as part of an engineering design course. The final design was selected through an evaluation matrix focused on cost, efficiency, safety, and ease of assembly.

    Door Alarm System

    Designed and implemented a door intrusion detection system using the STM32 Nucleo board. The system uses an IR break beam sensor and an ultrasonic sensor to detect door activity and object presence, activating a buzzer and LEDs as visual and audible alerts. Real-time feedback is provided through USART communication, with a push button included for manual system reset.
